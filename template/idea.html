Movie Recommendation NLP Project

Welcome to our Movie Recommendation NLP project! In this project, we aim to provide personalized movie recommendations to users based on their interests and preferences.
We have implemented a content-based filtering techniques.

Content-Based Filtering

Content-based filtering is a recommendation method that is based on the characteristics of items.
In our project, we use various features of movies, such as genre, description, cast, etc., to generate recommendations.
The similarity between movies is calculated using cosine similarity.

Cosine Similarity

Cosine similarity is a measure of the similarity between two non-zero vectors.
In our project, we use cosine similarity to measure the similarity between movies based on their features.
Given two movie feature vectors, the cosine similarity between them will be high if the angle between the vectors is small, indicating that the movies are similar.

Vectorization

In our project, we have used the 5000 most frequent words in the dataset as features for our movie descriptions.
This means that each movie description has been transformed into a 5000-dimensional vector, where each dimension represents the presence or absence of one of the 5000 words.

Data

The data utilized in this project was sourced from Kaggle, specifically the 5000 Movie TMDB dataset file.

How to use

To use the system, simply input a list of movie titles that you have liked in the past.
Our system will generate a list of movies using content-based filtering techniques.
